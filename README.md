# 万能文書作成・デザインツール

プレスリリース、公的文書、ビジネス文書など、様々な種類の文書を簡単に作成・デザインできるWebアプリケーションです。

## 主な機能

### 📄 文書作成機能
- **リアルタイムプレビュー**: 編集内容が即座にプレビューに反映
- **複数セクション対応**: 見出しと本文のセクションを自由に追加・削除・並び替え
- **改ページ機能**: 任意の位置で改ページを挿入可能
- **太字フォーマット**: `**テキスト**` で簡単に太字を適用
- **画像挿入**: メイン画像の追加とサイズ調整

### 🎨 デザインカスタマイズ
- **10種類のレイアウト**:
  1. **標準** - バランスの取れた汎用レイアウト
  2. **公的文書** - 日本の公的機関で使用される標準フォーマット（MS明朝、字下げ、適切な行間）
  3. **モダン** - グラデーションヘッダーを使った現代的なデザイン
  4. **シンプル** - ミニマルで読みやすいレイアウト
  5. **エレガント** - 明朝体を使った格式高いデザイン
  6. **アクセント** - 強調された左ボーダーが特徴
  7. **ビジネス** - 企業向けの信頼感あるフォーマット
  8. **クリエイティブ** - グラデーションタイトルと独創的なデザイン
  9. **ニュース** - 新聞スタイルのクラシックなレイアウト
  10. **アカデミック** - 学術論文風の端正なフォーマット

- **フォント選択**:
  - 標準フォント: Noto Sans JP、Noto Serif JP
  - UDフォント: BIZ UDPゴシック、BIZ UDP明朝
  - MS系フォント: MS Pゴシック、MS P明朝、メイリオ、遊ゴシック

- **カラーカスタマイズ**: テーマカラーをグラデーションで自由に設定
- **サイズ調整**: タイトル、見出し、本文のフォントサイズを個別に調整

### 💾 データ管理
- **自動保存**: ブラウザのローカルストレージに自動保存
- **履歴機能**: 過去に作成した文書を保存・読み込み
- **データエクスポート/インポート**: JSON形式でデータをバックアップ・共有

### 📤 エクスポート機能
- **PDF出力**: 高品質なPDFファイルとして保存
- **画像出力**: PNG/JPG形式で画像として保存
- **複数ページ対応**: 改ページを含む複数ページ文書のPDF出力

## 使い方

### 基本的な文書作成の流れ

1. **基本情報の入力**
   - タイトル、発行日、識別ID、発行者名を入力

2. **レイアウトとデザインの選択**
   - 10種類のレイアウトから用途に合わせて選択
   - フォント、カラー、サイズを調整

3. **コンテンツの作成**
   - 「セクション追加」ボタンで見出しと本文を追加
   - 必要に応じて「改ページ挿入」で新しいページを作成
   - 太字にしたい部分は `**テキスト**` で囲む

4. **画像の追加（オプション）**
   - 画像設定から画像ファイルを選択
   - サイズスライダーで調整

5. **エクスポート**
   - 「PDF保存」ボタンでPDFとして出力
   - または「画像」メニューからPNG/JPGで保存

### 便利な機能

#### 太字ボタン
各セクションの入力欄上部にある「太字」ボタンを使うと、選択したテキストを自動的に `**` で囲んで太字にできます。

#### 発行者名のカスタマイズ
発行者名のドロップダウンから「-- 新規追加 --」を選択すると、カスタムの発行者名を保存できます。

#### 履歴の活用
「作成履歴」セクションで過去の文書を管理できます。
- 履歴項目をクリックして読み込み
- 「現在の内容を保存」で新しい履歴を作成
- 不要な履歴は削除ボタンで削除

#### データのバックアップ
「データ」メニューから:
- 「JSON保存」でデータをファイルとして保存
- 「読み込み」で保存したJSONファイルを復元
- 「キーをコピー」でデータキーをクリップボードにコピー

## 技術仕様

### 使用技術
- **HTML5/CSS3/JavaScript**: フロントエンド
- **html2canvas**: プレビューのキャプチャ
- **jsPDF**: PDF生成
- **localStorage**: データの永続化
- **Font Awesome**: アイコン
- **Google Fonts**: Webフォント

### ブラウザ対応
- Chrome/Edge (推奨)
- Firefox
- Safari

### データ保存
すべてのデータはブラウザのlocalStorageに保存されます。ブラウザのデータを削除すると、保存した内容も失われますのでご注意ください。重要なデータは定期的にJSON形式でエクスポートすることをお勧めします。
