# 生徒会お知らせ作成ツール (Student Council Announcement Tool)

生徒会のお知らせや広報紙を簡単に作成し、PDFとして書き出すためのツールです。
赤を基調としたデザインですが、色は自由にカスタマイズ可能です。

## 特徴
- **リアルタイムプレビュー**: 入力した内容がすぐに右側のA4プレビューに反映されます。
- **デザインカスタマイズ**:
  - テーマカラー（生徒会カラーの赤など）
  - フォント（ゴシック、明朝、丸ゴシック）
  - レイアウト（標準、モダン、シンプル）
- **PDF書き出し**: ワンクリックでA4サイズのPDFファイルを生成します。
- **画像挿入**: メイン画像をアップロードし、サイズを調整できます。

## 使い方
1. このフォルダ内の `index.html` をダブルクリックしてブラウザ（Chrome推奨）で開いてください。
2. 左側のサイドバーにある各項目を入力・変更してください。
3. 画像がある場合は「画像設定」からアップロードしてください。
4. 調整が終わったら、左下の「PDFとして保存」ボタンを押してください。

## 動作環境
- インターネット接続（フォントやアイコンの読み込みに必要です）
- 推奨ブラウザ: Google Chrome, Microsoft Edge

## ファイル構成
- `index.html`: ツールの本体
- `style.css`: デザインスタイルシート
- `script.js`: 動作ロジック
